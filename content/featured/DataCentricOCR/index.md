Pure data-centric approach for receipt text detection with fixed EAST architecture.

## Key Achievements

| Phase          | F1 Score     | Improvement    | Innovation            |
| -------------- | ------------ | -------------- | --------------------- |
| **Baseline**   | 0.20         | -              | Raw dataset           |
| **Optimized**  | **0.8321**   | **+315%**      | Data-centric AI       |
| **Efficiency** | Same quality | **50% faster** | Pipeline optimization |

## Technical Highlights

- Applied data-centric AI principles to improve OCR performance without changing model architecture
- Enhanced dataset quality through advanced preprocessing and augmentation techniques
- Used Albumentations library for sophisticated image transformations
- Improved F1 score from 0.20 to 0.8321 (+315% improvement) through data optimization
- Optimized inference pipeline for 50% faster processing while maintaining accuracy
- Demonstrated the power of data-centric approaches over model-centric improvements
